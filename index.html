What is MindBoxed?

As the title indicates, the MindBoxed is a tile-based game with allows players to create levels. The basic mechanism is just picking pre-designed tiles and implementing that to the actual level. What makes it different is the community system, the reward system from the community and "Best of the Day" such social features. The purpose of the project is to let kids learn the basic method for designing and let them feel a sense of achievement by receiving external recognition.

How did we build it??

We started building the project by brainstorming some ideas we had, after deciding on a general topic, we began to write some documentation for the project and outlined different steps and sections we had to complete. We created a Website template to introduce and demonstrate the project. The other main tool we used was Unity, we've utilized it for basic functionality and believe that we can meet our future needs via this engine We operated on Aseprite and procreate to create pixel-style artistry to use as sprites in the game.

What is the future of it?

In order to make the game more interesting, there are several branches that we can consider. To make the production process more creative and offer more chances, we are trying to add more possible obstacles to the existing games; speeding up, traps, and doors that only allow to pass through fixed times. To make the game mot restricted to one mode of just passing the stage, it is feasible if we add more modes to the existing game; speed-racing, puzzles, rush mode and challenges, to add to the variability of the game. For the last part, if the game needs more social features and brings the creators a sense of achievement, we will add a special limited trap if someone created one level which was played by more than 10 people and get a liked rate of more than 70%.
